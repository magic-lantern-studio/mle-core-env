This directory contains builds of the Magic Lantern Android Studio for the Windows XP/Vista/7 platforms.

Contents:

mleandroidstudio-1.0.03132012-32bit-alpha-windows-installer.exe is the BitRock InstallBuilder installation package for the 32-bit Alpha release.

mleandroidstudio-1.0.02162012-alpha-windows-installer.exe is the BitRock InstallBuilder installation package for the 64-bit Alpha release.

magic-lantern-installation.pdf is the installation documentation for the current Alpha release.
